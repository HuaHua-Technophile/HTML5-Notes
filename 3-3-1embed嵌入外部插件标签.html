HTML中的 embed 标签是用于在HTML文档中嵌入外部应用程序的，通常是多媒体内容，如音频或视频。它也可以作为嵌入插件，如flash动画的容器¹²。
而chrome浏览器不支持该标签，可以使用iframe标签来代替


这个标签是HTML5中的一个新标签，它只需要开始标签，不需要结束标签¹⁴。

type属性的属性值有：application audio font example image message model multipart text video
